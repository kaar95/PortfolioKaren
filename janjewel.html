Overview

Jane's Jewelry is an independent jewelry and accessories brand. It manages lines for all women, from the most modern to the classics.

Problem

The brand's website has too many usability problems, it is not intuitive and several of the functionalities are missing to make it a user-friendly website.

Goal

-Increase Jane's Jewelry sales.

-Connect customers with the brand efficiently.

-Make purchasing products online easier.

Possible Solution

The proposal is to create additional screens and extra functionalities to solve the main problems of the website, which will be tentatively included:

-More efficient filters for products.

-Error screens that explain the problem the user is facing.

-Simplify the main menu.

-Add option for favorites.

My Role

  • Prototipyng

Tools

  • Figma
  • Adobe Illustrator

Painpoints

Error message

The users did not get any clear error message, we created a error screen with the type of error displayed so the clients can have a beeter understanding of the problem.

Filter

Users were having a hard time finding the products since the filter only had one option (Price), we added two other categories and the option to organize the content (Sort by).

Favorites

It was evident that users preferred to have a quick option to find the products they had previously seen without having to go to the shopping cart. For this reason, a new functionality was implemented to add favorites.
